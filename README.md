# 🎫 Ticket Tamer Tribe

Sistema moderno de gerenciamento de tickets desenvolvido com **React**, **TypeScript**, **Vite** e **Tailwind CSS**.

## 🚀 Sobre o Projeto

O Ticket Tamer Tribe é uma aplicação web para controle e acompanhamento de chamados (tickets), ideal para equipes de suporte técnico, atendimento ao cliente e operações internas.

A solução oferece uma interface moderna, responsiva e intuitiva, permitindo gerenciar solicitações de forma organizada e eficiente.

---

## ✨ Funcionalidades

- ✅ Criação e edição de tickets
- ✅ Acompanhamento de status
- ✅ Priorização de chamados
- ✅ Interface responsiva
- ✅ Componentização com React
- ✅ Tipagem estática com TypeScript
- ✅ Estilização com Tailwind CSS

## ⚙️ Instalação

```bash
git clone https://github.com/maksyflay-code/ticket-tamer-tribe.git
cd ticket-tamer-tribe
npm install
```

---

## ▶️ Executando o Projeto

```bash
npm run dev
```

A aplicação ficará disponível em:

```text
http://localhost:5173
```

---

## 📦 Build para Produção
